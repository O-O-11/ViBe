### 공모전 주제 : AI활용 차세대 교육 솔루션
주요 내용 : 교육 현장에서 발생하는 다양한 페인 포인트를 AI 기술로 해결하는 솔루션을 기획 및 구현  
- 가이드라인 : 
    본 공모전은 단순한 온라인 교육 플랫폼(LMS) 구축에 국한되지 않습니다. 
    교강사, 수강생, 교육 운영자 등 **교육 현장 구성원들의 실질적인 문제**를 날카롭게 해결하는 AI솔루션을 지향합니다.

- 최종 제출물 : 아래의 제출물을 본 메일로 제출
    1) 공모작 Git-Hub 저장소 주소 (반드시 public 설정, API Key 노출되지 않도록 주의)
    2) 배포된 라이브 URL
    3) AI 리포트 (첨부된 양식에 작성 후 PDF 파일로 첨부)
    4) 개인정보 수집/이용 동의서 및 참가 각서 (서명 완료된 PDF 파일)
