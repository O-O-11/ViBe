## 🚀 5분 안에 시작하기

### Step 1: 환경 준비 (1분)

```bash
# 프로젝트 최상위 폴더에서
cd ViBe
```

### Step 2: 의존성 설치 (2분)

**Windows:**
```bash
setup.bat
```

**macOS/Linux:**
```bash
bash setup.sh
```

**또는 수동으로:**
```bash
# 백엔드
cd backend
npm install

# 프론트엔드 (다른 폴더라면)
cd ../frontend
npm install
```

### Step 3: 서버 실행 (1분)

```bash
# ViBe 최상위 폴더에서

# Windows
npm start

# macOS/Linux
npm start
```

**콘솔 출력:**
```
🚀 ViBe 비디오 회의 서버 시작: http://localhost:3000
```

### Step 4: 브라우저에서 테스트 (1분)

1. 브라우저 열기: `http://localhost:3000`
2. 이름 입력 (예: "테스트유저1")
3. "새로 생성" → "강의실 생성"
4. 새 탭에서 같은 URL 접속
5. 이름 입력 (예: "테스트유저2")
6. 첫 번째 탭의 방 ID 복사
7. 두 번째 탭에 방 ID 입력 → "참여"
8. ✅ 화상통화 시작!

## 🔍 문제 해결

### 카메라/마이크 접근 권한 없음
- 브라우저 주소창의 🔒 아이콘 클릭
- "사이트 설정" → "카메라/마이크" → "허용"

### 연결 안됨
- 콘솔 열기 (F12)
- 에러 메시지 확인
- Socket.IO 연결 상태 확인

### 음성 안들림
- 다른 탭/앱이 마이크를 사용 중이지 않은지 확인
- 브라우저 재시작
- 시스템 음량 확인

## 📋 빠른 참조

### 단축키
- `Enter` - 채팅 메시지 전송 (입력 중일 때)

### 트러블슈팅

| 문제 | 해결책 |
|------|--------|
| "카메라/마이크 접근 권한 오류" | 브라우저에서 카메라/마이크 권한 허용 필요 |
| "비디오가 안 나옴" | 브라우저 주소창의 카메라/마이크 아이콘 확인 |
| "화면 공유가 안 됨" | HTTPS 또는 localhost 환경 필요 |
| "다른 사용자 비디오가 안 보임" | 네트워크 연결 확인, STUN 서버 접근성 확인 |
| "서버 연결 안 됨" | `npm start` 실행했는지 확인, 포트 3000 수용 가능 확인 |

### 포트 변경
```bash
# .env 파일 생성 (필요시)
PORT=8080

# 또는 직접 실행
PORT=8080 npm start
```

## 📊 기술 명세

### 서버 아키텍처
```
Express Server (localhost:3000)
├── Static Files (public/)
├── Socket.IO Namespace (/)
│   ├── join-room
│   ├── offer/answer/ice-candidate
│   ├── screen-share events
│   ├── chat
│   └── leave-room
└── 활성 사용자/방 관리
```

### 클라이언트 아키텍처
```
Web Browser
├── WebRTC PeerConnections (1:N)
├── Socket.IO Client
├── Local Media Stream
├── Remote Media Streams
└── UI Controllers
```

### 데이터 흐름
```
User A                  Server                  User B
  |                       |                       |
  |----join-room--------->|                       |
  |                       |<-----join-room--------
  |                       |
  |----offer------------>|-----offer------------>
  |                       |
  |<----answer-----------|<-----answer---------
  |<--ice-candidate------|<--ice-candidate-----
  |                       |
  |<===== WebRTC Stream (P2P) =====>
  |                       |
  |----chat-message----->|-----chat-message---->
  |                       |
```
